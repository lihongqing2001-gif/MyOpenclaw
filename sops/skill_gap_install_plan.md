# SOP: 技能缺口检查与安装计划

## Goal
针对一个目标 SOP 或目标任务，检查当前缺少哪些技能、能力包或集成，并输出安装与验证计划。

## Default Trigger
- “跑这个 SOP 还缺什么”
- “帮我看要装哪些技能”
- “给这个工作流做一份安装计划”

## Inputs
- 目标 SOP 名称或节点 ID
- 目标自动化等级

## Preconditions
- 可以读取技能树与当前技能目录
- `find-skills` 或已有技能索引可用

## Output Contract
默认输出：
- `outputs/skill_gap_plan_<目标>.md`

至少包含：
- 已具备能力
- 缺失能力
- 安装命令
- 验证步骤
- 优先级建议

## Steps
1) 确定目标 SOP
2) 读取现有技能与依赖
3) 匹配缺失项
4) 生成安装命令和验证清单
5) 给出最短落地顺序

## Failure Handling
- 如果目标 SOP 不存在，先给出相近节点候选
- 如果技能来源不明确，优先输出验证步骤而不是假定已安装
