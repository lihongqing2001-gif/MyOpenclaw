# SOP: 项目资料交接包

- Capability ID: cap.openclaw.sop.workflow-d522a60e
- Node ID: sop-项目资料交接包
- Domain: 项目产出与资料管理
- Area: 文件与资料管理

## Summary
把一个目录、项目或仓库整理成可以直接交给别人继续接手的交接包。

## Preconditions
目录可访问；`文件与版本管理` 可用；允许写入目标目录或 `outputs/`

## Inputs
- 目标目录 (optional)
- 交接对象 (optional)
- 完整度等级 (optional)

- “给这个项目做一份交接包”: 把一个目录、项目或仓库整理成可以直接交给别人继续接手的交接包。
