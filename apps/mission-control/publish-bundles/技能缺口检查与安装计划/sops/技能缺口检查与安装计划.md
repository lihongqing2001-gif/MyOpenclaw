# SOP: 技能缺口检查与安装计划

- Capability ID: cap.openclaw.sop.workflow-9fe1238c
- Node ID: sop-技能缺口检查与安装计划
- Domain: OpenClaw Workflow
- Area: 综合工作流

## Summary
针对一个目标 SOP 或目标任务，检查当前缺少哪些技能、能力包或集成，并输出安装与验证计划。

## Preconditions
可以读取技能树与当前技能目录；`find-skills` 或已有技能索引可用

## Inputs
- 目标 SOP 名称或节点 ID (optional)
- 目标自动化等级 (optional)

## Required Capabilities
- `cap.openclaw.skill.find-skills` · find-skills · bundled
