# SOP: 博客文章分析与改写方案

- Capability ID: cap.openclaw.sop.workflow-c62a80e1
- Node ID: sop-博客文章分析与改写方案
- Domain: 社交媒体与内容
- Area: 内容采集

## Summary
对一篇博客文章做结构、观点、风格与可传播性分析，并输出改写建议或多平台转化方案。

## Preconditions
文章可访问；允许写入 `outputs/`

## Inputs
- 博客链接或文章正文 (optional)
- 分析目标 (optional)
- 是否要改写 (optional)
