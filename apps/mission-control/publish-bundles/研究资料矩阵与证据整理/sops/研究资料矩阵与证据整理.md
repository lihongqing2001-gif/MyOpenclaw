# SOP: 研究资料矩阵与证据整理

- Capability ID: cap.openclaw.sop.workflow-ff3ad013
- Node ID: sop-研究资料矩阵与证据整理
- Domain: 科研与研究
- Area: 研究工作流

## Summary
围绕一个研究问题或项目主题，整理来源、观点、证据、风险和下一步检索方向，形成一份可继续扩展的研究矩阵。

## Preconditions
`web-search-plus` 可用；`qmd` 可用于知识检索；允许写入 `content_system/01_research/` 或 `outputs/`

## Inputs
- 研究主题 (optional)
- 关键问题 (optional)
- 时间范围 (optional)
- 输出路径（可选） (optional)
