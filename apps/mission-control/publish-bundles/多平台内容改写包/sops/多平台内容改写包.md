# SOP: 多平台内容改写包

- Capability ID: cap.openclaw.sop.workflow-4fd5a428
- Node ID: sop-小红书内容多平台改写包
- Domain: 社交媒体与内容
- Area: 内容改写

## Summary
把一份已有素材或一篇已有成品，改写成多平台可直接发布的内容包。

## Preconditions
源材料可访问；`content_system/final/` 可写；如需风格对齐，可调用已有内容风格知识

## Inputs
- 源材料路径或正文 (optional)
- 目标平台 (optional)
- 风格要求 (optional)
- 输出数量 (optional)

## Required Capabilities
- `cap.openclaw.skill.social-content` · social-content · bundled

- “把这篇内容改成多平台版本”: 把一份已有素材或一篇已有成品，改写成多平台可直接发布的内容包。
- “给我一份小红书 + 视频口播 + 朋友圈的改写包”: 把一份已有素材或一篇已有成品，改写成多平台可直接发布的内容包。
- “基于这份素材出一套内容发布包”: 把一份已有素材或一篇已有成品，改写成多平台可直接发布的内容包。
