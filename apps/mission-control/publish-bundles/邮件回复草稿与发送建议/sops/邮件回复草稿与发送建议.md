# SOP: 邮件回复草稿与发送建议

- Capability ID: cap.openclaw.sop.workflow-8c040ba6
- Node ID: sop-邮件回复草稿与发送建议
- Domain: 个人管理
- Area: 日程规划与提醒

## Summary
针对一封或一组邮件生成可直接使用的回复草稿，并给出发送建议。

## Preconditions
邮件可读取；如果要自动发送，必须单独授权

## Inputs
- 邮件主题或邮件 ID (optional)
- 回复目标 (optional)
- 回复语气 (optional)

## Required Capabilities
- `cap.openclaw.skill.assistant-orchestrator` · assistant-orchestrator · bundled
- `cap.openclaw.skill.email-sequence` · email-sequence · bundled

## Use Cases
- “帮我回这封邮件”: 针对一封或一组邮件生成可直接使用的回复草稿，并给出发送建议。
- “给这几封邮件写回复草稿”: 针对一封或一组邮件生成可直接使用的回复草稿，并给出发送建议。
- “整理成可发送版本”: 针对一封或一组邮件生成可直接使用的回复草稿，并给出发送建议。
